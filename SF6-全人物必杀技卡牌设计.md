# 街头霸王6 全人物必杀技卡片设计

> 基于 Fighter 角色 mod 现有机制设计。所有卡牌与**帧数净值（FrameAdvantage）**、**打康（CounterHit）**、**连击（Combo）**、**格斗精神（FightingSpirit）** 系统联动。
> 本文档仅给出费用、伤害/格挡、机制设计，不含代码实现。

---

## 现有机制速查

| 机制 | 说明 |
|------|------|
| **帧数** | 正负净值资源。正帧可消耗打出强力技能；负帧时受到攻击伤害+20% |
| **打康** | 攻击意图为"攻击"的敌人时，伤害+20%，获得2帧 |
| **确反康** | 完全格挡敌人攻击后，下一次攻击伤害+20%，获得4帧 |
| **轻击** | 获得一层连击。连击：上限一层，与有利帧同时消耗时可免费打出重击牌，回合结束时消失|
| **重击** | 消耗X帧和一层连击时可免费打出|
| **斗气** | 每场战斗初始6层，每回合回复一层，最高6层。有斗气时+1力量/+1敏捷，归零则Burnout（2层虚弱+2层易伤），且无法获得斗气，两回合后获得6层斗气 |
| **投技** | 无视防御造成伤害 |
| **超必杀** | 打出后消耗所有有利帧，本回合每打出一张攻击牌伤害提高10% |
---

## 卡牌稀有度说明

| 稀有度 | 范围 | 定位 |
|--------|------|------|
| **基础 (Basic)** | 角色初始卡组 | 简单直接，数值扎实 |
| **普通 (Common)** | 常规掉落 | 标准强度，无特殊条件 |
| **罕见 (Uncommon)** | 较少掉落 | 有条件效果，套路核心 |
| **稀有 (Rare)** | 稀有掉落 | 强力效果，需资源联动 |

# 设计理念总结

## 角色机制分布

| 角色类型 | 角色 | 核心机制 |
|----------|------|----------|
| **帧数管理型** | Ryu, Ken, Luke, Cammy, Rashid, Ed | 通过攻击牌获取帧数，再用帧数打出更强技能 |
| **投技型** | Zangief, Manon, Alex, Lily | 高伤害指令投，消耗帧数，无视格挡 |
| **蓄积资源型** | Jamie (酒量), Juri (杀界), Lily (风衣), Ingrid (太阳徽记), A.K.I. (毒), Bison (地雷), JP (裂隙) | 用技能牌蓄积资源，再用大招消耗 |
| **蓄力/防守反击型** | Guile, Blanka, Honda, Dee Jay, Chun-Li, Marisa | 蓄力或格挡后打出更强力的攻击 |
| **玻璃大炮型** | Akuma | 极高伤害但自损防御 |
| **多连击/多段型** | Chun-Li, Honda, Rashid, Elena | 多段攻击，配合打康和帧数系统 |
| **派生选择型** | Dee Jay (Jus Cool), Cammy (Hooligan), Ed (Kill Rush), Elena (Lynx Song), Sagat (Tiger Nexus), C. Viper, Alex (Prowler) | 单卡多派生，提供灵活战术选择 |

## 数值设计原则

| 稀有度 | 伤害/费用比 | 说明 |
|--------|-------------|------|
| 基础 | ~4:1 标准值 | 有基础加成+3 |
| 普通 | ~5:1 标准值 | 有机制联动可上浮至6-7:1 |
| 罕见 | ~6:1 基础值 | 条件达成可到8-9:1 |
| 稀有 | ~7-9:1 + 特殊效果 | 需要资源/条件支撑 |

以上所有数值均为初步设计，实际实现时需要根据游戏内部平衡调整。
